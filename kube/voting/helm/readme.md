# Helm

## Install Helm

```bash
helm install {name} {chart}

helm install nginx ./nginx-chart
```

## Upgrade Helm

```bash
helm upgrade {name} {chart}

helm upgrade nginx ./nginx-chart
```

## List Helm

```bash
helm list
```

## Uninstall Helm

```bash
helm uninstall {name}

helm uninstall nginx
```
